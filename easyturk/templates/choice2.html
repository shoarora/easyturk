Instructions

  • In this task, we will present a text, a list of possible synonyms, and a label.
  • For each synonym option, select whether it is different than, similar to, or the same as the text.

  • Same is defined as:
    • option can be replaced by text and maintain the same meaning.
    • note that this doesn't mean they have to be fully interchangeable. replacing option with text should maintain the same meaning, but the reverse doesn't need to be true.
    • you could think about this in the context of a google search. would replacing option with text give you the same results as option? Think about this in the context of label.
  • Similar is defined as: the option is similar to the text, but would slightly change the meaning if replaced.
  • Different is defined as: option and text have different intended meanings.

  • When in doubt, please refer to the examples below, do a quick Google search, or contact us. We'll work to clarify the instructions.
